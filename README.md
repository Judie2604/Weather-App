🌦️ Weather App
A simple Weather App that fetches real-time weather data using an API and displays it with a user-friendly UI built using HTML, CSS, and JavaScript.

🚀 Features
✔️ Fetches real-time weather data using an API (e.g., OpenWeatherMap)
✔️ Displays temperature, humidity, and weather conditions
✔️ Dynamic background/icons based on weather conditions
✔️ User-friendly and responsive UI

🛠️ Technologies Used
HTML – Structure of the app
CSS – Styling and layout
JavaScript – Fetching API data and updating UI
Weather API – Fetching real-time weather details
